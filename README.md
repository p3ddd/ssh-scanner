# ssh-scanner

## Introduction

ssh-scanner is a tool for scanning machines on a network by SSH.

## Usage

  - `ssh-scanner <prefix> <start> <end> <user> <password>`
  - `ssh-scanner <prefix> <user> <password>`
  - `ssh-scanner <prefix>`
  - `ssh-scanner <subnet> <user> <password>`
  - `ssh-scanner <subnet>`
